Lukas' StarNotary Token DApp

Contract address: 0x5EB5AFf80B9e9119E12FFB5f847758a99D1caC18 (Goerli)

StarNotary implements an ERC721 token representing newly discovered stars and the star name. When a new star(token) is minted, a name is given to the star.

The star tokens can be traded according to ERC721 specifications. A sale is achieved if a seller sets up a star for sale, with an asking price and a buyer subsequently calls the buy function with at least that price.


Create a Star

This is used to mint new stars. Only authorized addresses are allowed to mint.







Sell a star

You can put up your star for sale here. As soon as a buyer accepts the asking price, ownership will be transferred.







Buy a star

This is used to buy stars. If the star you're trying to sell is not put up for sale, you'll get an error message.
